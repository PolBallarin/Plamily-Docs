# Bienvenido a Plamily

Plamily es una aplicación diseñada para facilitar el cuidado y gestión de personas mayores. Permite a familias y cuidadores coordinar medicamentos, citas, contactos importantes y más, todo desde un solo lugar.

## Funcionalidades principales

- **Gestión de medicamentos**: Programa recordatorios y lleva un control de las tomas.
- **Calendario**: Organiza citas médicas y eventos importantes.
- **Agenda de contactos**: Guarda información de médicos, cuidadores y otros contactos relevantes.
- **Multi-hogar**: Gestiona diferentes residencias desde una sola cuenta.
- **Permisos**: Invita a familiares y cuidadores con diferentes niveles de acceso.
- **Notificaciones personalizables**: Configura alertas según tus necesidades.

## Primeros pasos

1. Descarga la app desde App Store o Google Play.
2. Crea tu cuenta con email y contraseña.
3. Configura tu primera residencia.
4. Añade los mayores que vas a cuidar.
5. ¡Empieza a gestionar medicamentos y citas!
